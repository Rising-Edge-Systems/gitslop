## Codebase Patterns
- Typecheck: `npm run typecheck` (runs typecheck:node + typecheck:web)
- CommitGraph.tsx uses react-window v2 with `useListCallbackRef` for list virtualization
- CSS modules pattern: `styles.className` with `CommitGraph.module.css`
- HEAD commit identified by `refs.some(r => r.type === 'head')` — type 'head' means the current branch ref
- Lane 0 is always HEAD lane; lane colors via `color-mix()` CSS function
- Floating overlays: position absolute within `.container`, z-index 1200+ range
- `listRef.scrollToRow({ index, align })` for programmatic scrolling
- Graph nodes computed via `computeGraphLayout(commits)` → `useMemo`
- All repo-modifying git ops go through `withWatcherSuppression()` in git-ipc.ts — it handles watcher suppression + forced refresh
- `sendRepoChangedForced()` in index.ts bypasses watcher suppression for explicit repo:changed events
