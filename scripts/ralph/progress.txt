## Codebase Patterns
- Sidebar: plain flex div (NOT react-resizable-panels), width in pixels via layout.sidebarSize
- Detail panel: plain flex div (NOT react-resizable-panels), fixed 340px width, always visible
- Right panel layout: DetailPanel on top + StatusPanel on bottom, stacked in flex column
- Staging area (StatusPanel): now renders in the RIGHT panel, not the center panel
- Center panel: only commit graph + diff viewer (no staging area)
- CSS modules: import styles from './Component.module.css', camelCase class names
- Design tokens: src/renderer/src/styles/tokens.css — --space-*, --font-*, --radius-*, --border, --bg-hover, --transition-fast
- Typecheck: npm run typecheck
- Build: npx electron-vite build && npx electron --no-sandbox .
- Layout state: useLayoutState() hook, localStorage key 'gitslop-layout-state'
- State threading: AppLayout → MainContent → RepoView (layout state only in AppLayout)
- Tab system: useRepoTabs() in App.tsx, TabBar.tsx component
- Lucide icons: import { IconName } from 'lucide-react'
- react-resizable-panels: ONLY for vertical split (center+terminal). NOT for sidebar or detail panel.
- DetailPanel: hooks must ALL be above any early return to avoid React error #310
- Use the frontend-design skill for any UI changes — invoke it to get design guidance
