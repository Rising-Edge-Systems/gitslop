## Codebase Patterns
- Typecheck uses `npm run typecheck` (split into `typecheck:node` and `typecheck:web` via tsconfig.node.json / tsconfig.web.json)
- Tests use `npm test` (vitest)
- Main process is monolithic in `src/main/index.ts` with IPC handlers using `ipcMain.handle()`
- electron-store schema defined in `StoreSchema` interface, defaults in `new Store<StoreSchema>({ defaults: {...} })`
- Auto-updater already existed with basic setup; IPC channels now use `updates:*` prefix
- BrowserWindow.getAllWindows()[0] pattern for sending events to renderer
