## Codebase Patterns
- TypeScript typecheck: `npm run typecheck` (runs both `tsconfig.node.json` and `tsconfig.web.json`)
- Tab state management is in `src/renderer/src/hooks/useRepoTabs.ts` — tabs persist via localStorage
- When `activeRepoPath` is null, MainContent shows WelcomeScreen; when set, shows RepoView
- TabBar is rendered in App.tsx between TitleBar and ErrorBoundary
- CSS modules are co-located with components (e.g., `TabBar.module.css` alongside `TabBar.tsx`)
- CSS uses design tokens: `var(--bg-secondary)`, `var(--border)`, `var(--text-muted)`, `var(--transition-fast)`, etc.
- DetailPanel accepts `detail: CommitDetail | null` — null shows empty state, non-null shows commit info
- DetailPanel `onFileClick` callback passes `(file, commitHash)` for center-stage diff integration
- AppLayout always renders the detail Panel inline when `currentRepo` is set (no overlay mode)
- Detail panel collapse: controlled via `detailPanelCollapsed` in LayoutState, collapsed strip is a 24px div outside the react-resizable-panels Group
- react-resizable-panels `Group` only accepts `Panel` and `Separator` children — put non-panel elements outside the Group in a flex wrapper
- Center-stage diff state (viewingDiff/diffFile/diffCommitHash) lives in AppLayout, threaded through MainContent → RepoView
- `window.electronAPI.git.showCommitFileDiff(repoPath, hash, filePath)` loads diff content for DiffViewer
- Hover/active states: use `var(--bg-hover)` and `var(--bg-active)` tokens (defined in tokens.css)
- Section header convention: `var(--font-sm)`, `font-weight: 600`, `text-transform: uppercase`, `letter-spacing: 0.5px`, `color: var(--text-muted)`
- Use `color-mix(in srgb, var(--color) N%, transparent)` for semi-transparent color variants from tokens
- Global focus-visible rule in global.css covers all interactive elements — component-level can still override
